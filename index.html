Portfolio

React Google Books Search
Find your next favorite book!

Users can search the Google Books API to find a book, and then save it to the database for later viewing!

Tech used: React, Google Books API, MongoDB, Express, Node.js

The Relocator
Find your forever home!

The Relocator is an app for users that want to compare two cities. The app compares the location, cost of living, and weather data of two cities. The Relocator also stores previous user searches for later use.

Tech used: W3 CSS Framework, JQuery, Open Weather API, Numbeo API, OpenStreetMap API, MapBox, Day.js

Dank Reviews
Review Memes from Reddit!

Dank Reviews lets users create an account and review silly memes from Reddit. It uses the reddit.images NPM package to pull a random meme. Users can see other users reviews and all of the reviews are stored in a MySQL database.

Tech used: Reddit.images API, Sequelize, Bcrypt, DOTENV, MySQL2, Express, Handlebars, Session, MySQL

Online/Offline Budget Tracker
Track your expenses online and offline!

This online/offline budget tracker uses IndexedDB to store user input data offline, so when the user is back online, the IndexedDB data is added to the MongoDB.

Tech used: IndexedDB, MongoDB, Javascript, Express

Employee Directory
Filter and sort random employees!

An employee directory application built in React. The application pulls between 10-20 users using the Random User Generator API. You can then filter and sort users by hire date, first name, last name, email and phone number.

Tech used: React.js, Random User Generator API, Javascript

Weather Dashboard
Check the weather!

A weather dashboard app that displays both the current weather and the five day forecast for a searched city. The The UV index is color coded based on the severity level. The user's search history is saved in local storage.

Tech used: HTML5, Bootstrap, Javascript, Open Weather API
